# Standar SEO ## Prinsip Inti - Desain untuk pengguna terlebih dahulu, mesin pencari kedua - Buat konten berkualitas tinggi dan berharga - Pastikan keunggulan dan performa teknis - Bangun untuk pengguna seluler sebagai prioritas - Fokus pada metrik pengalaman pengguna - Terapkan data terstruktur untuk meningkatkan visibilitas - Pertahankan praktik pengoptimalan yang konsisten - Pantau dan sesuaikan dengan perubahan algoritme ## SEO Teknis ### Struktur Halaman - Terapkan struktur semantik HTML5 yang tepat - Gunakan satu tag H1 per halaman - Buat hierarki judul yang logis (H1 → H2 → H3) - Sertakan judul yang deskriptif dan kaya kata kunci - Gunakan elemen HTML semantik (`<article> Bahasa Indonesia:<section> Bahasa Indonesia:<nav> `, dll.) - Terapkan markup schema.org untuk tipe konten - Pertahankan konten penting dalam HTML, bukan JavaScript - Pastikan konten dapat diakses tanpa JavaScript ### Tag Meta - Buat tag judul yang unik dan deskriptif (50-60 karakter) - Tulis deskripsi meta yang menarik (120-155 karakter) - Terapkan tag kanonik untuk mencegah konten duplikat - Gunakan atribut bahasa yang tepat (`<html lang="en"> `) - Terapkan tag meta Open Graph dan Twitter Card - Sertakan tag meta viewport untuk desain responsif - Gunakan tag meta robots dengan tepat - Terapkan atribut rel (nofollow, ugc, sponsored) untuk tautan ### URL - Buat URL yang pendek dan deskriptif - Gunakan tanda hubung untuk memisahkan kata - Sertakan kata kunci yang relevan jika natural - Pertahankan struktur situs yang logis - Terapkan penanganan parameter URL yang tepat - Hindari parameter URL jika memungkinkan - Gunakan huruf kecil di URL - Hindari karakter khusus di URL ### Performa - Raih skor 90+ dalam metrik performa Lighthouse - Optimalkan Core Web Vitals: - LCP (Largest Contentful Paint): &lt; 2,5 detik - FID (First Input Delay): &lt; 100 md - CLS (Cumulative Layout Shift): &lt; 0,1 - Terapkan pengoptimalan gambar yang tepat - Minimalkan CSS dan JavaScript - Manfaatkan cache browser - Aktifkan kompresi (Gzip, Brotli) - Kurangi waktu respons server - Terapkan CSS penting - Gunakan lazy loading untuk gambar dan video ### Pengindeksan &amp; Perayapan - Buat dan kelola XML sitemap - Terapkan file robots.txt - Siapkan Google Search Console dan Bing Webmaster Tools - Pantau statistik dan kesalahan perayapan - Perbaiki tautan rusak dan kesalahan 404 - Terapkan kode status yang tepat - Gunakan rel=&quot;alternate&quot; untuk varian bahasa/regional - Kontrol anggaran perayapan untuk situs besar ## Optimasi Konten ### Strategi Kata Kunci - Lakukan riset kata kunci menyeluruh - Fokus pada maksud pencarian daripada kepadatan kata kunci - Targetkan kata kunci berekor panjang untuk konten tertentu - Kelompokkan kata kunci terkait menurut topik - Analisis kata kunci pesaing - Pertimbangkan kata kunci SEO lokal jika relevan - Lacak peringkat dan kinerja kata kunci - Perbarui strategi kata kunci secara teratur ### Pembuatan Konten - Buat konten asli dan berharga - Pastikan panjang konten yang sesuai untuk topik - Sertakan kata kunci target secara alami - Gunakan istilah dan sinonim terkait - Struktur konten dengan judul dan subjudul yang jelas - Sertakan konten visual (gambar, video, infografis) - Perbarui konten secara teratur - Hapus atau tingkatkan konten yang tipis atau ketinggalan zaman ### Optimasi Pada Halaman - Sertakan kata kunci dalam elemen penting: - Tag judul - Judul H1 - URL - Paragraf pertama - Subjudul (jika alami) - Optimalkan gambar dengan nama file deskriptif dan teks alt - Terapkan tautan internal dengan teks jangkar deskriptif - Buat teks jangkar deskriptif untuk tautan eksternal - Gunakan daftar isi untuk konten yang panjang - Terapkan skema FAQ untuk konten berbasis pertanyaan - Optimalkan untuk cuplikan unggulan - Sertakan elemen ajakan bertindak ## Pengoptimalan Seluler - Terapkan desain responsif - Pastikan navigasi ramah seluler - Optimalkan target ketuk (minimal 44×44px) - Hilangkan sumber daya yang menghalangi render - Minimalkan beban sumber daya untuk perangkat seluler - Uji pada beberapa perangkat seluler - Pastikan ukuran font dapat dibaca tanpa memperbesar - Hindari interstisial yang mengganggu ## Pengalaman Pengguna - Minimalkan rasio pentalan dengan konten yang menarik - Tingkatkan kecepatan pemuatan halaman - Pastikan navigasi intuitif - Terapkan remah roti - Buat ajakan bertindak yang jelas - Pastikan aksesibilitas (kepatuhan WCAG) - Minimalkan iklan dan popup yang mengganggu - Optimalkan untuk waktu tunggu dan keterlibatan ## SEO Lokal - Buat dan optimalkan Profil Bisnis Google - Pastikan konsistensi NAP (Nama, Alamat, Telepon) - Terapkan markup skema bisnis lokal - Buat halaman khusus lokasi - Optimalkan untuk kata kunci lokal - Dorong ulasan pelanggan - Daftarkan bisnis di direktori yang relevan - Buat konten lokal ## SEO E-commerce - Terapkan markup skema produk - Buat deskripsi produk yang unik - Gunakan gambar produk berkualitas tinggi dengan teks alt - Sertakan ulasan pelanggan - Optimalkan halaman kategori - Tangani produk yang kehabisan stok dengan benar - Terapkan navigasi breadcrumb - Buat URL produk yang dioptimalkan ## Pemantauan &amp; Analisis - Siapkan pelacakan analitik yang tepat - Pantau lalu lintas pencarian organik - Lacak rasio konversi dari pencarian organik - Analisis metrik perilaku pengguna - Pantau peringkat kata kunci - Lakukan audit SEO teknis rutin - Lacak profil backlink - Pantau kinerja pesaing ## SEO Lanjutan ### SEO Internasional - Terapkan tag hreflang untuk penargetan bahasa/wilayah - Gunakan struktur URL yang sesuai (ccTLD, subdomain, atau subdirektori) - Buat konten lokal - Pertimbangkan hosting lokal untuk kinerja yang lebih baik - Terapkan penargetan geografis di Search Console - Gunakan tag meta bahasa yang tepat - Pertimbangkan perbedaan budaya dalam strategi SEO ### SEO JavaScript - Pastikan konten terlihat dalam HTML yang dirender - Terapkan rendering dinamis jika diperlukan - Gunakan implementasi lazy loading yang tepat - Uji dengan alat &quot;Lihat Sumber yang Dirender&quot; - Minimalkan rendering sisi klien untuk konten penting - Terapkan kode status yang tepat untuk navigasi JavaScript - Uji dengan Uji Ramah Seluler Google ### Pencarian Suara Optimasi - Optimalkan untuk kata kunci percakapan - Buat konten FAQ - Fokus pada cuplikan unggulan - Tingkatkan kecepatan halaman - Optimalkan untuk kueri lokal - Gunakan data terstruktur - Tulis dengan nada percakapan - Fokus pada kata kunci pertanyaan (siapa, apa, kapan, di mana, mengapa, bagaimana)
